# AMS - Attendance Management System

This is a modern, premium monorepo Attendance Management System (HRMS) built using React Native (Expo) and a Node.js/Prisma/MongoDB API backend.

## Project Structure
- `apps/mobile`: React Native (Expo) mobile frontend
- `apps/api`: Node.js, Prisma, and MongoDB backend
- `packages/ui`: Shared design system and components
- `packages/shared-types`: Shared TypeScript typings
